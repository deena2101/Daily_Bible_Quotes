# Daily_Bible_Quotes
A modern bilingual devotional web application that delivers inspirational Bible verses in English and Malayalam through a calm, elegant, and interactive user interface.

# Features
-Bilingual support (English & Malayalam)
-Interactive quote reveal cards
-Randomized quote shuffle
-Elegant dark spiritual UI with subtle animations
-Fully responsive design

# 🎯 Purpose
This project was built to create a simple, distraction-free devotional experience where users can reflect on Bible verses in a peaceful digital environment.

# 🛠️ Tech Stack
-HTML5
-CSS3
-JavaScript
